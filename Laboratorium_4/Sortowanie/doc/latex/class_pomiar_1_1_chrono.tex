\-Stworzona do pomiarów czasu. 

\-Klasa służy do mierzenia czasu wykonywania metod. \-Możliwe jest uzyskanie czasu działania w\-:\par

\begin{DoxyItemize}
\item milisekundach, \par

\item mikrosekundach, \par

\item nanosekundach. \par
 \-Dodatkowo klasa umożliwia zapisywanie wyników pracy do plików o nazwie \char`\"{}\-Wyniki\-\_\-temp.\-txt\char`\"{}, które analizowane są przez program do bechmarku.
